Nixie-tube clock
----------------

Created by Andras Dobozi
Eszterhazy Karoly University
2018

-----------------------------------------------------------------------------------
-----------------------------------------------------------------------------------

This is an ATMega328P microcontroller based clock.

-----------------------------------------------------------------------------------
-----------------------------------------------------------------------------------

Required parts:
---------------

4x Nixie-tube
4x 74141PC BCD to Decimal converter
12V -> 150V high voltage power supply
12V 3A power supply
Atmel ATMega 328P microcontroller
DS1307 RTC (real time clock) module
